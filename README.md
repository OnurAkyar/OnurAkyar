## 🧠 Projects

🎮 **Triversi Game (C)**  
Implementation of the Triversi strategy board game using the C programming language.  
The project focuses on algorithmic thinking, board state management, and game logic implementation.

📊 **Numerical Analysis Calculator (C)**  
A mathematical calculator capable of solving numerical analysis problems such as root finding and equation evaluation.  
The system parses mathematical expressions and implements numerical methods such as Bisection and other iterative techniques.

📚 **Library Automation System (C)**  
A file-based library management system developed using linked lists and dynamic memory.  
The system manages books, authors, and students, and supports borrowing operations.

✈️ **Airline Customer & Staff Management System (Java)**  
A management system designed to manage airline customers and personnel records.  
Includes operations such as customer tracking, staff management, and structured data handling.

📈 **Gradient-Based Optimization Algorithms (GD, SGD, Adam)**  
Implementation and comparison of different gradient-based optimization algorithms including:
- Gradient Descent (GD)
- Stochastic Gradient Descent (SGD)
- Adam Optimizer

The project analyzes convergence behavior, optimization efficiency, and performance differences between these algorithms in machine learning training tasks.

🤖 **Cosmos-Gpt2-Medium(Math)**  
A project focused on training and optimizing large language models using techniques such as:
- Supervised Fine-Tuning (SFT)
- Direct Preference Optimization (DPO)
- GRPO training strategies

The project includes dataset preparation, training pipelines, and evaluation scripts for model optimization.

🧠 **YKS-2026 AI Model Training Project**  
A project focused on training a specialized AI model for solving **YKS-style exam questions**.  
The project includes:
- Dataset preparation for YKS-type problems  
- Training pipelines for large language models  
- Preference-based training strategies  
- Evaluation of model performance on exam-style questions  

The goal is to build an AI model capable of understanding and solving university entrance exam problems effectively.
